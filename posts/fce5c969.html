由于如果直接使用java -jar 项目名.jar启动springboot项目，会造成你启动了这个项目就不用想输入其他命令了，所以使用 “nohup java -jar springboot项目名.jar >记录日志的文件名.log 2>&1 &”命令，来让spirngboot项目后台运行，并且记录日志。 如果不想输出日志，可以使用“nohup java -jar springboot项目名.jar >/dev/null 2>&1 &”
