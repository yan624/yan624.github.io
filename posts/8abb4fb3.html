js一些常用的属性和函数

发表于 2017-09-05 更新于 2020-06-09 分类于 IT-stuff ， front-end 阅读次数：
本文字数： 1k 阅读时长 ≈ 2 分钟

由于老是忘记js和jq的函数，这里记一下。 ### javascript 1. string.replace(//, "") 第一个参数是正则，表达式在//两个之间写，第二个参数是需要替换成什么。 2. string.substr() 用法和一样 3. new Date(date).toLocaleString() 将毫秒数转成本地的时间，如中国：2017/9/4 下午9:42:41 4. (a/b).toFixed(2) 相除保留两位小数，如700/100等于7.00。要不然就是只等于7（java里整数相除只得到整数部分）

Jquery

  1. $("#id").attr() 为一个节点添加属性，一个属性:attr("name", "file")。多个属性:attr({"name":"file", "type","button"})
  2. $("#id").remove() 移除节点
  3. $("#id").removeAttr("") 移除一个节点的属性
  4. $("#id").children(":first") 一个节点的子节点
  5. $("#id").offset() 一个节点离网页顶部的top值和left值，不是离浏览器的top值和left值（以整个网页的高度做计算）。 $("#id").offset().top代表节点的top值，left值同理。
  6. $("body,html").animate({ scrollTop:scroll_offset.top //让body的scrollTop等于pos的top，就实现了滚动 },0); 滚动监听
  7. window.location.search 获得当前url，如：http://localhost:8080/contextpath/index.html
  8. \(.parseJSON(string) 将一个字符串转为json对象，但是不要在用ajax的时候，data:\).parseJSON(string)这样用，我搞了2个小时，没查出错，原来是这里错了。 string json = '{"comment":"abcdrf","name":"xiaowang"}'; 我在用\(.ajax()的时候，把data赋值\).parseJSON(json)，出错了！！！data:json。后台接受的很成功。 但是后台传来一个json字符串，用这个方法转成json对象，可以直接通过.取值挺方便的。
  9. new FormData().append() 使用 FormData对象获取到form中的数据后，可以继续append自己想要添加的数据。
