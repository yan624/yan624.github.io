70. 爬楼梯

  这是道简单题，思想是找到递推式。简单
  题设你正在爬楼梯，需要 n 阶你才能到楼顶，那么我们反着思考，在此之前我们设 OPT(i) 代表 i 阶楼梯共有几种走法。现在假设现在只需要一步就可以到楼顶，我们只有两种选择：1）一步跨一阶；2）一步跨两阶。它们分别对应于 OPT(i - 1) 和 OPT(i - 2)，显然这两种走法是不同的！所以我们实际上 OPT(i) = OPT(i - 1) + OPT(i - 2)。这是其实已经可以用数组解决了，已知一阶楼梯只有一种走法，二阶楼梯有两种走法，则已知数组 [1, 2]，然后第三项只是前两项的和得到 [1, 2, 3]，以此类推得到 [1, 2, 3, 5]。

  这是道简单题，思想是找到递推式（2020.3.26：今天学到了，原来这公式叫状态转移公式）。简单
  首先要理解一点，这里的相邻房屋指的是字面意思，虽然这样理解在理论上有点说不通。比如说现在有 A，B，C 三幢房子，偷 AB 房子会触发警报，BC也会触发，但是 AC 不会。
  理解了相邻房子的概念之后，就可以做题了。显然这题是“偷与不偷”的想法，我们将所有房子用一个数列表示。设有 n 间房，则对于第 n 间房，我们可以选择“偷”和“不偷”。如果“偷”，则第 n-1 间房不能偷，下间可“偷”的房是第 n-2 间；如果“不偷”，则第 n-1 间房可以偷，下间可“偷”的房是第 n-3 间。我们假定 OPT(i) 表示可以偷到钱财的最大值，nums 表示各间房屋中所拥有的钱财。则“偷”可表示为 OPT(n) = OPT(n-2) + nums[n]，“不偷”可表示为 OPT(n) = OPT(n-3) + nums[n-1]。将表达式形式化为：
